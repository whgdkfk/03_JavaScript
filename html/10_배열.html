배열(Array)

        * 변수: 메모리상에 값을 저장하는 공간(값 1개 저장)

        * 배열: 변수를 묶음으로 다루는 것

        - 배열에 저장된 요소를 구분하기 위해서 index(색인) 이용
        (0부터 시작)

        - JS 배열 특징
        1. 자료형의 제한 X
        (여러 자료형 저장 가능, 하지만 보통 한 가지로 통일)

        2. 길이 제한 X
        (자동으로 배열의 길이가 늘었다, 줄었다 함)

        3. 배열을 위한 다양한 필드(변수)/메서드(함수)가 제공됨
        ex) 배열.length -> 배열 길이 반환
            배열.indexOf('값') -> '값'이 몇 번 인덱스 요소에 있는지 알려줌
                                    (없으면 -1)
    
        [JS 배열 선언 방법]
        
        1. const arr = [];
        // 0칸 짜리 배열 생성
        * []: 배열 기호

        2. const arr = new Array();
        // 0칸 짜리 배열 생성
        * new 연산자: 새로운 객체 생성

        3. const arr = new Array(5);
        // 5칸 짜리 배열 생성
        // -> 처음 생성될 때는 5칸 모두 비어있음(empty)

        4. const arr = [값1, 값2, 값3];
        // 3칸 짜리 배열 생성
        // -> 각 인덱스번째 요소에 값이 자동 대입
        // 0:값1, 1:값2, 2:값3

배열 X for문

        - 배열에 저장된 요소들은 index를 이용해서 구분함

        - index는 
        0부터 시작해서
        배열 길이 -1까지
        1씩 증가하는 규칙성을 보임
        (index는 무조건 0 이상의 정수(소수, 실수X))

        - index 값을 증가시켜 배열 내 요소를 접근할 때
        가장 효율적인 방법이 for문을 이용하는 것!

2차원 배열(n차원 배열)

        * 변수 : 메모리상에 값을 저장하는 공간(값 1개 저장)
        ex) const num1 = 10;
        ex) const num2 = 20;
        ex) const num3 = 30;

        * 1차원 배열 : 변수를 묶음으로 다루는 것(변수 n개)
        ex) const arr1 = [10, 20, 30];
            const arr2 = [40, 50, 60];
            const arr3 = [70, 80, 90];


        * 2차원 배열 : 1차원 배열을 묶음으로 다루는 것(1차원 배열 n개)
        ex) const arr = [
                            [10, 20, 30],
                            [40, 50, 60],
                            [70, 80, 90]
                        ];
        
            arr[0][2] === 30;
            arr[2][1] === 80;            

        * 3차원 배열 : 2차원 배열을 묶음으로 다루는 것(2차원 배열 n개)
