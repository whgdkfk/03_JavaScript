DOM 요소 생성 / 추가 / 제거

        [생성]
        1) document.createTextNode("내용") -> TEXT 노드 생성
        2) document.createElement("태그명") -> HTML 요소 생성

        *** 주의사항 ***
        위 구문 수행 시 노드/요소가 생성만 된 것이지
        아직 HTML 문서(DOM)에 추가되지 않은 상태이다.

        [추가]
        // append: (글에) 덧붙이다, 첨부하다
        1) 요소.appendChild(노드/요소)
           -> 요소의 마지막 자식으로 노드/요소를 추가

        2) 요소.append(요소 [, 요소2, 요소3, ...]) <-> prepend
           -> 요소의 마지막 자식으로 () 내 모든 요소를 추가
           * () 내 요소가 추가될 때 1, 2, 3... 순서대로 추가된다. 

        3) 요소.prepend(요소 [, 요소2, 요소3, ...]) <-> append
           -> 요소의 첫 번째 자식으로 () 내 모든 요소를 추가
           
        4) 요소.before(요소)
           -> 요소의 이전 형제로 추가

        5) 요소.after(요소)
           -> 요소의 다음 형제로 추가

        [제거]
        1) 요소.remove()
           -> 지정된 요소를 제거
