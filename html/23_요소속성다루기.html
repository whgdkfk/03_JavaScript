요소 속성(Attribute) 관련 메서드

        1. 요소.getAttribute("속성명"): 속성 값 얻어오기

        2. 요소.setAttribute("속성명", "속성값")
           -> 요소에 새로운 속성 추가
              (이미 같은 속성이 존재하면 덮어쓰기)
        
        3. 요소.removeAttribute("속성명"): 속성 제거하기

        4. 요소.hasAttribute("속성명"): 속성 존재 여부 확인(T/F)

        // DOM: HTML을 자바스크립트 객체로 만들어 놓은 것
        - DOM 요소에 노출되어 있는 속성은

        1) 짧고 간결함
        2) Attribute() 관련 메서드보다 속도적 우위를 가진다.
        3) 표준 HTML에 대해 직접 접근 가능하다.

class 관련 속성(classList)

        * class 속성의 특징
        1) 모든 HTML 요소가 가질 수 있는 속성
        2) 중복 가능
        3) 한 요소가 여러 클래스를 가질 수 있다.

        [tip]
        - attribute: HTML 요소 속성
        - property: JS 객체 속성

        * 요소.classList
        - 요소의 class를 조작하기 위한 속성(property)
        
        1) 요소.classList.add("클래스1"[, ...])
           -> 요소에 클래스를 추가(여러 클래스 한 번에 추가 가능)
           -> 요소에 class가 존재하고 있으면 이어서 추가(덮어쓰기 X)

        2) 요소.classList.remove("클래스1"[, ...])
           -> 요소에 클래스를 제거(여러 클래스 한 번에 제거 가능)

        3) 요소.classList.toggle("클래스명") // toggle: 둘 중 하나 선택
           -> 요소에 클래스가 있으면 제거, 없으면 추가
        
        4) 요소.classList.contains("클래스명") 
           -> 요소에 클래스가 포함되어 있으면 true, 없으면 false(반복문, 조건문에서 사용)
        
        5) 요소.classList.replace("기존 클래스명", "새 클래스명")
           -> 요소가 가지고 있는 "기존 클래스명"을 "새 클래스명"으로 변경
